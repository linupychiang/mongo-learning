## MongoDB 版本变迁

- 0.x 起步阶段 2008
- 1.x 支持复制集和分片集 2010
- 2.x 更丰富的数据库功能 2012
- 3.x WiredTiger 和周边生态环境 2014
- 4.x 分布式事务支持 2018

## MongoDB vs 关系型数据库

|              | MongoDB                                                      | 关系型数据库           |
| ------------ | ------------------------------------------------------------ | ---------------------- |
| 数据模型     | 文档模型                                                     | 关系模型               |
| 数据库类型   | OLTP                                                         | OLTP                   |
| CRUD 操作    | MQL/SQL                                                      | SQL                    |
| 高可用       | 复制集                                                       | 集群模式               |
| 横向扩展能力 | 通过原生分片完善支持                                         | 数据分片或者应用侵入式 |
| 索引支持     | B-树、全文索引、地理位置索引、多键（multikey）索引、TTL 索引 | B 树                   |
| 开发难度     | 容易                                                         | 困难                   |
| 数据容量     | 没有理论上限                                                 | 千万、亿               |
| 扩展方式     | 垂直扩展+水平扩展                                            | 垂直扩展               |
